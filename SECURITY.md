# Security Policy

Security policy for `saebooks-web` is managed centrally with the
engine repo. See
<https://github.com/saebooks/saebooks/blob/master/SECURITY.md>.

## TL;DR

- Email <security@saee.com.au> with reproduction steps and impact.
- Do **not** open a public GitHub issue for vulnerabilities.
- We acknowledge within 72 hours and coordinate disclosure.
