{# Reusable pagination control with page numbers. When `extra_query` is supplied it should NOT include a leading `&` or `?` — the macro joins it correctly. Pass "" if you have no extra params. Showing {{ offset + 1 }}–{{ [offset + limit, total] | min }} of {{ total }} · page {{ current }} of {{ total_pages }}
