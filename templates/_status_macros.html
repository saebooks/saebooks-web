{# Shared status-badge macros. `payment_status_badge(item, today, due_soon_cutoff)` — derives the visible badge for any bill / invoice / credit-note-like row that has: .status (DRAFT | POSTED | VOIDED) .amount_paid (Decimal/number) .total (Decimal/number) .due_date (ISO 8601 date string or null) Routes pass `today` and `due_soon_cutoff` (today + 7 days) as ISO strings; YYYY-MM-DD compares lexically so we can avoid date parsing in Jinja. Mapping: VOIDED → grey "Voided" DRAFT → amber "Draft" POSTED + amount_paid >= total → green "Paid" POSTED + amount_paid > 0 → blue "Partial" POSTED + due_date < today → red "Overdue" POSTED + due_date <= today + 7d → amber "Due soon" POSTED + everything else → indigo "Open" #} {%- macro payment_status_badge(item, today=None, due_soon_cutoff=None) -%} {%- set s = (item.status or '')|upper -%} {%- if s == 'VOIDED' -%} {{ _("Voided") }} {%- elif s == 'DRAFT' -%} {{ _("Draft") }} {%- elif s == 'POSTED' -%} {%- set total = (item.total or 0)|float -%} {%- set paid = (item.amount_paid or 0)|float -%} {%- if total > 0 and paid >= total -%} {{ _("Paid") }} {%- elif paid > 0 -%} {{ _("Partial") }} · {{ money(paid, grouping=False) }} / {{ money(total, grouping=False) }} {%- elif today and item.due_date and item.due_date < today -%} {{ _("Overdue") }} {%- elif due_soon_cutoff and item.due_date and item.due_date <= due_soon_cutoff -%} {{ _("Due soon") }} {%- else -%} {{ _("Open") }} {%- endif -%} {%- else -%} {{ s|capitalize }} {%- endif -%} {%- endmacro %} {# `simple_status_badge(item)` — for entities with no payment state (credit notes, expenses, quotes). Just maps DRAFT/POSTED/VOIDED to colour. #} {%- macro simple_status_badge(item) -%} {%- set s = (item.status or '')|upper -%} {%- if s == 'POSTED' -%} {{ _("Posted") }} {%- elif s == 'VOIDED' -%} {{ _("Voided") }} {%- elif s == 'DRAFT' -%} {{ _("Draft") }} {%- else -%} {{ s|capitalize }} {%- endif -%} {%- endmacro %}
